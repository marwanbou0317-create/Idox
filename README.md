# Facebook Messenger Bot

بوت فيسبوك ماسنجر للمجموعات — مبني بـ Node.js

## الأوامر

| الأمر | الوصف |
|-------|-------|
| `/ping` أو `/بينج` | اختبار البوت |
| `/اوامر` | عرض قائمة الأوامر |
| `/سيرفر` | معلومات السيرفر |
| `/ابتيم` | وقت تشغيل البوت |
| `/محرك` | تشغيل/إيقاف المحرك |
| `/محرك وقت [ثواني]` | تحديد وقت المحرك (min: 10ث) |
| `/محرك رسالة [نص]` | تحديد رسالة المحرك |
| `/محرك الذكي` | الإرسال عند النشاط فقط |
| `/محرك حالة` | حالة المحرك |
| `/قفل` | قفل/فتح المجموعة |
| `/قفل الذكي` | قفل كل المجموعات |
| `/كنية @شخص [كنية]` | تعيين كنية |
| `/كنية الكل [كنية]` | كنية لكل الأعضاء |
| `/كنيات [نص]` | نفس الكنية للجميع |
| `/رفع @شخص` | رفع لمشرف (سوبر أدمن فقط) |
| `/اخفاض @شخص` | إنزال من مشرف (سوبر أدمن فقط) |

## التشغيل

```bash
npm install
node index.js
```

## النشر على Railway

1. ارفع المشروع على Railway
2. سيتم التشغيل تلقائياً عبر `railway.toml`

## تجديد الجلسة

إذا انتهت الجلسة:
1. سجّل دخول فيسبوك في المتصفح
2. صدّر الكوكيز بـ Cookie-Editor (صيغة JSON)
3. استبدل محتوى `appstate.json`
4. أعد تشغيل البوت
